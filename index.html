What is Pinhead-Font?

Quality public domain icons for your map pins

Pinhead has 1,000+ icons and counting, including standardized versions of the most popular public domain cartographic icon sets: Maki, Temaki, OSM Carto, and NPMap.

How to use it?

You can use Pinhead-Font as a font or as SVG symbols or images.
Please refer to the Pinhead repository for SVG files.

To use it in a web page, just add the css in your project. 

<link href="https://viglino.github.io/pinhead-font/css/pinhead-font.css" rel="stylesheet" />

Then use an inline element with a class prefixed with pin- to add a new icon. 

<i class="pin-pin"></i>

Using a <span> is more semantically correct but a little bit verbose: 

<span class="pin-bicyclist"></span>

Or use it as an svg sprite (svg sprites are inlocated in the ./dist/pinhead-font.svg file): 

<svg class="pinhead-font pin-2x">
  <use xlink:href="path/to/dist/pinhead-font.svg#pin-polyline-pt" />
</svg>

Icon size and color

To increase icon sizes relative to their container, use the pin-lg (33% increase), pin-2x, pin-3x, pin-4x, or pin-5x classes.

Use CSS color to change font color:

Fixed width icons

Use pin-fw class to set icons at a fixed width. Especially designed to use in nav lists & list groups to preserve alignment.

  • Bus with destination
  • Compact box truck
  • Bicycle in locker
  • Jet plane

Rotated and flipped icons

Use pin-rotate classes to rotate icons (-20, -10, 10, 20, 45, 90, 135, 180, 225, 270 or 315).

Use pin-flipv or pin-fliph classes to flip icons vertically or horizontally.

Stacked icons

To stack multiple icons, use the pin-stack class on the parent. 

<span class="pin-stack pin-3x">
  <i class="pin-circle_outline" style="color:#666;"></i> 
  <i class="pin-diamond_tall pin-rotate45" style="color:#800;"></i> 
</span>
